# dijango 使用

### 安装dijango
```
    pip3 install dijango
```

### 创建 app
```
    dijango-admin startproject appName
```

### 部署 app
```
    python manage.py runserver 8080
```

#### 部署 到 nginx


### 创建admin 后台
```
    python manager.py makemigrations
    python manager.py migrate
    python manager.py createsuperuser
    
```
